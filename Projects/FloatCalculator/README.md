# FLoat Calculator
A calculator that works with 4 elemetry operations(+, -, *, /) and float numbers
each operation in one separate line


## Scripts
 1. run.sh: <br />
    compile, link and run the calculator <br />
    ```bash
    cd <REPOSITORY_ROOT>/Projects/FloatCalculator/ && source run.sh
    ```
 
 2. run.sh: <br />
    compile, link the calculator and run it with gdb <br />
    ```bash
    cd <REPOSITORY_ROOT>/Projects/FloatCalculator/ && source run-with-gdb.sh
    ```
